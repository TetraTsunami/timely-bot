Timekeeper

What is Timekeeper?

Timekeeper is a Discord bot for managing times for users across the world using Discord's timestamp feature. Once a user has set up their time zone with the bot, it will monitor their posts. If the user posts anything with a time in it, the bot will reply with that time converted to a timestamp, which will display as the correct local time for whoever views it.

Times will be recognized if they are formatted like 8:30 PM, 8pm, or 8:30. The bot can convert more than one time in a post, but it will ignore posts with more than 4 time entries to avoid spamminess.

A time zone may also be specified (e.g. 6:00 PM EST). Timekeeper currently recognizes the following time zones: EST/EDT, CST/CDT, MST/MDT, PST/PDT, and UTC.

Using Timekeeper

In a server that has the bot, any user can set up their time zone and daylight savings info by using the /timekeeper command. This will show the user a pair of dropdowns for selecting their settings.

After a user has entered their settings, then the bot will start working! Whenever that user makes a post, the bot will check it to see if it includes times. If any times are found, it will reply to the post with those times as a timestamp!
