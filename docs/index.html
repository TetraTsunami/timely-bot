Timekeeper

What is Timekeeper?

Timekeeper is a Discord bot for converting times for users across the world using Discord's timestamp feature. All you have to do to start using the bot is mark the time as code using backticks (`).

The bot may be used without any other configuration if you supply the time zone when writing a time - e.g. 3 PM EST, 8:30 PDT, 13:00 UTC, and so on. You can tag as many time entries as you like in a post and Timekeeper will convert them all.

The currently supported time zones are as follows: EST/EDT, CST/CDT, MST/MDT, PST/PDT, and UTC.

Automatic Time Conversion

If you would like to use the bot for local times without having to specify your own time zone, you may use the /timekeeper command to select your own time zone and daylight savings settings.

Once you have set yourself up, you may omit the time zone from any times you tag and Timekeeper will convert them from your local time - e.g. 3 PM, 8:30
